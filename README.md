# Drone Components Configurator & Price Calculator

Веб-додаток на Python (Streamlit) для розрахунку вартості комплектуючих дронів та літаків з інтеграцією Supabase.

## 🚀 Особливості

- **Адмін-панель**: Управління компонентами, масове редагування цін, видалення позицій
- **Гостьовий режим**: Вибір готових пресетів або кастомна збірка
- **Dark Mode UI**: Сучасний темний інтерфейс
- **Динамічний розрахунок**: Автоматичний перерахунок загальної вартості
- **Експорт в Excel**: Завантаження замовлення в форматі XLSX
- **Модульна архітектура**: Готовність до інтеграції з парсинг-агентом

## 📋 Вимоги

- Python 3.10 або новіше
- Аккаунт Supabase (безкоштовний план підійде)
- Веб-браузер з підтримкою JavaScript

## 🛠️ Встановлення

### 1. Клонуйте репозиторій або скопіюйте файли

```bash
git clone <repository-url>
cd pricesss
```

### 2. Створіть віртуальне середовище

```bash
python -m venv venv

# На Linux/Mac:
source venv/bin/activate

# На Windows:
venv\Scripts\activate
```

### 3. Встановіть залежності

```bash
pip install -r requirements.txt
```

### 4. Налаштуйте Supabase

#### 4.1. Створіть проєкт на Supabase

1. Перейдіть на [supabase.com](https://supabase.com)
2. Створіть новий проєкт
3. Дочекайтеся ініціалізації бази даних

#### 4.2. Виконайте SQL-схему

1. Відкрийте SQL Editor в Supabase Dashboard
2. Скопіюйте вміст файлу `supabase_schema.sql`
3. Виконайте запит (Run)
4. Перевірте, що таблиця `components` створена

#### 4.3. Отримайте API ключі

1. Перейдіть в Settings → API
2. Скопіюйте:
	- `Project URL` (URL)
	- `anon/public` ключ (API Key)

### 5. Налаштуйте Secrets

#### Локальна розробка

Створіть файл `.streamlit/secrets.toml`:

```bash
mkdir -p .streamlit
nano .streamlit/secrets.toml
```

Додайте наступний вміст:

```toml
# Supabase Configuration
[supabase]
url = "https://your-project-id.supabase.co"
key = "your-anon-key-here"

# Admin Configuration
[admin]
# Пароль: admin123 (змініть хеш для власного паролю)
# Для генерації хешу використайте:
# python -c "import hashlib; print(hashlib.sha256('your_password'.encode()).hexdigest())"
password_hash = "240be518fabd2724ddb6f04eeb1da5967448d7e831c08c8fa822809f74c720a9"
```

**Важливо**: 
- Замініть `your-project-id` на ID вашого Supabase проєкту
- Замініть `your-anon-key-here` на ваш anon key
- Для зміни пароля згенеруйте новий хеш:

```python
import hashlib
password = "ваш_новий_пароль"
hash_value = hashlib.sha256(password.encode()).hexdigest()
print(hash_value)
```

#### Деплой на Streamlit Cloud

1. Перейдіть в Settings → Secrets на Streamlit Cloud
2. Додайте той самий вміст у форматі TOML

## 📖 Використання

### Для гостей

1. Відкрийте додаток
2. Оберіть режим:
	- **Готовий пресет**: виберіть тип дрона (7-дюймовий камікадзе, 5-дюймовий FPV, тощо)
	- **Кастомна збірка**: оберіть компоненти вручну з фільтрацією
3. Додайте компоненти до кошика
4. Переглядайте загальну вартість в реальному часі
5. Експортуйте замовлення в Excel

### Для адміністраторів

1. Увійдіть через бічну панель (пароль за замовчуванням: `admin123`)
2. Доступні функції:
	- **Додати компонент**: заповніть форму та додайте новий товар
	- **Редагувати ціни**: масове редагування через інтерактивну таблицю
	- **Видалити компонент**: оберіть та видаліть непотрібні позиції
	- **Синхронізація**: функція для майбутнього парсинг-агента

## 🗄️ Структура бази даних

### Таблиця `components`

| Поле | Тип | Опис |
|------|-----|------|
| id | BIGSERIAL | Унікальний ідентифікатор |
| name | VARCHAR(255) | Назва компонента |
| type | VARCHAR(50) | Тип: квадрік / літак / перехоплювач |
| category | VARCHAR(100) | Категорія (мотори, ESC, рама, тощо) |
| price_uah | DECIMAL(10,2) | Ціна в гривнях |
| stock_status | BOOLEAN | Наявність на складі |
| url | TEXT | Посилання на магазин |
| created_at | TIMESTAMP | Дата створення |
| updated_at | TIMESTAMP | Дата оновлення |

## 🔮 Майбутня інтеграція парсинг-агента

Код підготовлений для інтеграції автоматичного парсера цін. Функція `sync_prices()` в файлі app.py містить архітектуру для:

1. Виклику зовнішнього сервісу парсингу
2. Оновлення цін з URL кожного компонента
3. Логування історії синхронізацій
4. Звітування про оновлені ціни

Для активації потрібно розробити окремий парсинг-сервіс або інтегрувати бібліотеки (BeautifulSoup, Selenium).

## 🎨 Кастомізація

### Зміна кольорової схеми

Редагуйте змінну `CUSTOM_CSS` в app.py

### Додавання пресетів

Змініть словник `DRONE_PRESETS` в app.py:

```python
DRONE_PRESETS = {
	 "Назва пресету": {
		  "type": "квадрік",
		  "categories": ["мотори", "рама", "ESC"]
	 }
}
```

### Зміна мови інтерфейсу

Всі текстові рядки знаходяться безпосередньо в коді. Для локалізації створіть окремий файл `locales.py`.

## 🐛 Усунення неполадок

### Помилка підключення до Supabase

- Перевірте правильність URL та API ключа
- Переконайтеся, що RLS політики налаштовані
- Перевірте стан проєкту на Supabase Dashboard

### Компоненти не відображаються

- Переконайтеся, що SQL-схема виконана повністю
- Перевірте наявність тестових даних в таблиці
- Відкрийте Developer Tools (F12) для перевірки помилок

### Не працює авторизація

- Перевірте правильність хешу паролю в `secrets.toml`
- Згенеруйте новий хеш та оновіть конфігурацію
